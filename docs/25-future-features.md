# 25 — Future Features

> Status: **Backlog / Idea pool.** Anything here is **out of MVP scope** and not actively worked on.

## Mobile

- React Native apps (customer + worker).
- PWA install prompts.
- Offline mode for workers (cached job list).

## Realtime

- Socket.io chat (1:1, customer ↔ worker).
- Typing indicators + read receipts.
- File attachments in chat.
- Push notifications (Web + Mobile).

## Maps & Geo

- Map view of nearby workers.
- Distance-based ranking.
- Route to job (turn-by-turn deep links).
- Geofencing for "in this area" filters.

## Payments

- Stripe Connect for worker payouts.
- Razorpay for India.
- Held escrow until job confirmed.
- Instant payout (with fee).

## Monetization Enhancements

- Featured listings for workers.
- Subscription tiers.
- Promoted categories.
- Sponsored search.

## Trust & Safety

- Background check integrations.
- ID verification (Aadhaar, PAN equivalents).
- Dispute resolution workflow with admin queue.
- Insurance offerings for high-value jobs.

## Productivity

- Recurring jobs (e.g., weekly cleaning).
- Job templates.
- Worker calendar + availability.
- Auto-quote suggestions.

## Analytics

- Cohort retention dashboards.
- Funnel drop-off analysis.
- Worker LTV.
- City heatmaps.

## AI / ML

- Smart matching (skill, proximity, rating).
- Job description auto-suggest.
- Fraud / anomaly detection.
- Review summarization.

## Team / Contractor

- Contractor accounts with multiple workers.
- Sub-account permissions.
- Invoicing + taxes.
- Bulk job handling.

## Internationalization

- Multi-language UI (Hindi, Spanish, etc.).
- Locale-aware formats.
- Currency display.

## Integrations

- WhatsApp Business notifications.
- Calendar sync (Google/Apple).
- CRM integration for admin.

## Platform / DX

- Public REST + Webhook APIs.
- Plugin marketplace (future).
- White-label offering.
