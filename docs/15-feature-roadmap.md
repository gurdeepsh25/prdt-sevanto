## Backlog (Prioritized)

- Email digest of new jobs.
- Saved workers / favorite list.
- Job expiry + auto-close.
- Worker availability calendar.
- Skills endorsements.
- Profile completeness scoring.
- Multi-language support.
- In-app banner notifications for admins.
- CSV export for admin reports.

## Future Backlog

- Mobile apps (React Native).
- PWA mode for workers.
- OAuth (Google, Apple).
- AI-assisted job description suggestions.
- Smart worker-customer matching.
- Team / contractor accounts.
- Subscription tiers for workers.
- Featured listings / sponsored placement.
- Platform commission engine.
- Tax forms / invoicing.
- Dispute resolution workflow.
- Background check integrations.
- Scheduled/repeat jobs.
